This document describes the processed datasets included in the package and how they were obtained. These files are cleaned, structured versions of the raw data stored in data_raw/. They are ready to be used inside the simulation tools for the homework.

  1. weather_processed.csv

Location: src/champagneSADEZLAN/data/processed_data/weather_processed.csv

Source: data_raw/weather_full.json (raw OpenWeather API data)

Generated by: data_raw/data_transformation.py

Description: This dataset contains cleaned weather information for a selection of world capital cities. Only variables relevant for the simulation model were retained:

Column Description
city Name of the city (string)
temperature Air temperature in Â°C (float)
humidity Relative humidity in % (float)
pressure Atmospheric pressure in hPa (float)

Cleaning steps performed:

Extracted "main" fields from the nested JSON structure.

Removed unused OpenWeather fields (wind, clouds, timestamps, etc.).

Converted any values stored in list format (e.g. [12.4]) into scalar floats.

Ensured consistent column names and numeric types.

Saved output as a flat CSV file.

Usage: This dataset is used to build CityWeather objects, which in turn allow the computation of:

expected guest count using expected_lambda(),

simulation inputs for party attendance.
